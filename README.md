# 🚀 QuickRozgar: Hyperlocal AI Talent Discovery

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://quickrozgar-final.tanmaykatiyar0207.workers.dev)
[![AI Powered](https://img.shields.io/badge/AI-Gemini%202.0%20Flash-blue.svg)](https://deepmind.google/technologies/gemini/)
[![Framework](https://img.shields.io/badge/Framework-TanStack%20Start-FF4154.svg)](https://tanstack.com/start)

**QuickRozgar** is a hyperlocal employment platform designed to bridge the gap between local workers and small businesses. By leveraging **Google Gemini AI**, QuickRozgar eliminates the friction of job posting and talent discovery, making "hiring next door" as simple as a single click.

---

## 🌟 Key Features

### 🧠 Gemini-Powered Matching Engine
Stop scrolling through generic resumes. Our **Smart Match** engine analyzes worker profiles and job requirements in real-time to provide human-like insights:
- *"Mike matches 90% of your requirements and lives only 1.2km away."*
- *"This role fits your weekend availability and pays 20% above average."*

### 🪄 AI Magic Extract
Creating job listings shouldn't be a chore. 
- Paste any messy text or voice-to-text description.
- Gemini automatically extracts the **Title, Salary, Location, and Shift details**.
- Go from "thought" to "live listing" in under 5 seconds.

### 📍 Hyperlocal Discovery
- **Visual Dashboards**: See jobs and workers in your immediate neighborhood.
- **Instant Invitations**: Employers can invite "Top Matches" to apply with one tap.
- **One-Click Apply**: Workers can apply for high-overlap jobs instantly.

---

## 🛠️ Tech Stack

- **Full-Stack Framework**: [TanStack Start](https://tanstack.com/start) (React + SSR)
- **AI Model**: [Google Gemini 2.0 Flash](https://deepmind.google/technologies/gemini/) (via Google Generative AI SDK)
- **Database & Auth**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with modern glassmorphism UI
- **Deployment**: [Cloudflare Workers](https://workers.cloudflare.com/) (Edge-ready)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/tanmaykatiyar0207-cell/quickrozgar-final
   cd quickrozgar-final
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file with:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_key
   VITE_GEMINI_API_KEY=your_google_gemini_api_key
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```
